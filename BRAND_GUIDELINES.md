# MirageAPI Brand Guidelines

## Trademark Usage

### Official Name
- **Correct**: MirageAPI
- **Incorrect**: Mirage API, mirage-api, MirageApi

### Logo and Wordmark
The MirageAPI wordmark uses:
- **Primary styling**: "Mi" and "API" in accent color (#a78bfa), "rage" in standard text
- **Font**: Display font family with specific letter spacing
- **Colors**: Purple accent (#a78bfa) with standard text colors

### Permitted Uses
- ✅ Referring to the software in documentation
- ✅ Educational content and tutorials
- ✅ Integration guides and API documentation
- ✅ Community discussions and support

### Restricted Uses
- ❌ Using MirageAPI name for derivative products
- ❌ Creating confusingly similar names
- ❌ Using the name to imply official endorsement
- ❌ Commercial use of the trademark without permission

### Attribution Requirements
When referencing MirageAPI:
- Include trademark notice: "MirageAPI™"
- Credit: "MirageAPI is a trademark of Satya Bandaru"
- Link to official repository or website when possible

### Domain and Web Presence
- **Official domain**: mirageapi.com
- **Repository**: github.com/venkatbandaru99/mirage
- **Trademark owner**: Satya Bandaru

### Contact
For trademark usage questions or permissions, contact the trademark owner.

---
**Note**: These guidelines protect the MirageAPI brand while allowing community use of the open-source software.
